# Dataset

Le dataset complet n'est pas inclus dans le dépôt GitHub, car certains fichiers CSV/PCAP dépassent largement les limites pratiques d'un dépôt Git.

Pour reproduire l'entraînement, récupérer le dataset MQTTset depuis la source utilisée dans le projet, puis placer les fichiers dans cette structure locale :

```text
dataset/
  Data/
    CSV/
    PCAP/
    FINAL_CSV/
```

Les modèles entraînés nécessaires à la démonstration sont conservés dans le dossier `models/`.
